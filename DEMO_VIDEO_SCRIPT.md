## 🛠️ Troubleshooting untuk Demo

### Jika Backend Tidak Jalan:
```
Gunakan data mock/placeholder di dashboard.
Focus pada UI/UX dan flow aplikasi.
```

### Jika Ada Error:
```
1. Refresh halaman (Cmd+R)
2. Restart dev server: npm run dev
3. Clear browser cache jika perlu
```

### Tips Recording:
```
- Gunakan screen recording software (OBS, QuickTime)
- Resolution: 1920x1080
- Frame rate: 30fps
- Audio: Clear narration
- Cursor: Highlight/highlighter effect
```
